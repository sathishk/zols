Zols has two distributions

  • Standalone - This is basic spring booted executable jar
  • Embeddable Jar - This is spring plug-in which we can include inside any Spring MVC application

However to begin with the basic features, it is easy to get started with standalone mode.

Start configuring the content by creating a new Category. The Category attributes consists of Name, Label and Description.

Links can be created within categories in a hierarchical fashion. The Link attributes are Name, Label, URL and Description.
